## PhotoStamp
Readme for PhotoStamp

### Hardware
#### Camera
|CAM No.|CAMx_OUT   |CAMx_IN    |
|-      |-          |-          |
|CAM1   |PD8        |PB13       |
|CAM2   |PD10       |PB11       |
|CAM3   |PD12       |PE15       |
|CAM4   |PD14       |PB13       |
|CAM5   |PC6        |PB11       |

#### Trigger
 - PWM_IN:  PE9
 - TRIG_IN: PC3

#### USART
 - GPS: PA3
 - Debug: PA9/PA10

#### Others
 - KEY: PA0 (Onboard)
 - LED: PA1 (Onboard)
